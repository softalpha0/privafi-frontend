# PrivaFi Frontend

React + TypeScript frontend for PrivaFi — a confidential DeFi suite built on Zama FHEVM.

For full project documentation, smart contracts, and deployment instructions see the main repo:
https://github.com/softalpha0/PrivaFi

## Run locally

```bash
npm install
npm run dev
```

Requires MetaMask connected to Sepolia testnet.
